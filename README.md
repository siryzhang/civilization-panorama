# 文明发展全景图 · Civilization Panorama

> 以马克思政治经济学为分析框架，横跨农业、工业、信息、AI 四个时代，系统梳理人类文明演化的底层逻辑。
>
>🔗Link： https://siryzhang.github.io/civilization-panorama/


---

## 这是什么

一张单文件交互式 HTML 全景图，从六个维度对四个文明时代进行结构性对比分析，并向 AI 时代纵向延伸，形成可横向比较、可纵向拓展的研究底座。

适合**战略研究者、组织设计者、HR 从业者、对文明演化感兴趣的思考者**作为思维框架使用。

---

## 框架结构

全景图以 **六维 × 四时代** 矩阵为核心：

| 维度 | 分析视角 |
|------|---------|
| **⚡ 生产力** | 核心驱动力、关键技术跃迁、效率量级 |
| **🏭 生产资料** | 核心资料构成、所有权结构、获取门槛、控制权逻辑 |
| **🔗 生产关系** | 人的关系、组织形态、组织分工、权力逻辑 |
| **💰 商业模式** | 核心逻辑、收入结构、代表性案例 |
| **⚖️ 价值创造与分配** | 创造机制、分配公式 |
| **🧭 组织第一性原理** | 核心命题、主导范式、人的管理职能、核心矛盾、失效模式 |

| 时代 | 时间范围 |

## 文档包含的延伸模块

除核心矩阵外，文档还包含三个延伸板块：

**① 六维演化规律与深层洞察**
每个维度独立的演化规律提炼，含 AI 时代推论：
- 生产力：跃迁加速·替代层级上移律
- 生产资料：所有权集中·新质垄断律
- 生产关系：人的自主性·解放-重捕螺旋律
- 商业模式：价值捕获·持续上游迁移律
- 价值分配：再平衡触发·被替代群体政治动员律
- 组织：核心威胁·持续内化律

**② 未来预测**
基于六维演化规律向前推演的六条预测，每条标注来源维度、时间窗口（2025–2035）、置信度与关键不确定性。

**③ AI 时代能力演化与构建**
- Part 1：9 个代表性科技岗位的演化路径（AI替代内容 / 人类剩余价值 / 进化后角色）
- Part 2：面向组织设计者与 HR 从业者的四步行动框架

## 分析框架来源

本全景图以**马克思政治经济学**的核心概念（生产力 / 生产资料 / 生产关系）为底层分析框架，并非意识形态表达，而是借助其结构性视角理解**价值如何在不同时代被创造、控制与分配**。

> "生产资料的所有权，决定了一个时代谁能从生产中获取最大收益。" 这一命题在农业时代、工业时代、信息时代、AI 时代均成立——只是生产资料的形态从土地→机器→数据平台→基础模型持续演化。
